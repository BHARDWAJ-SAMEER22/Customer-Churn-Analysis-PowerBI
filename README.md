## Project Overview
This project analyzes customer churn for a telecom company to identify key factors influencing customer attrition.

## Tools Used
- Power BI
- DAX
- Data Analysis

## Key KPIs
- Total Customers
- Churned Customers
- Churn Rate (%)

## Key Insights
- Month-to-month contracts have the highest churn.
- Customers using Electronic Check show higher churn.
- Fiber Optic users churn more than DSL users.
- Long-term contracts significantly reduce churn.

## Dashboard Pages
- Churn Analysis Dashboard
- Insights & Summary

## Objective
To help businesses understand customer churn patterns and improve retention strategies.
